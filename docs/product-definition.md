# Product Definition

## What It Is

Python for Kids is a beginner coding app where children learn Python by running
tiny missions and seeing what their code does in plain language.

## Target Learners

- Primary: ages 8-10
- Future: ages 5-7 with more guided editing and less typing

## Learning Philosophy

Children should see code as cause and effect:

```python
x = 1
x = x + 1
print(x)
```

Becomes:

1. Python remembers `x` as `1`.
2. Python changes `x` to `2`.
3. Python says `2`.

The app should teach real Python, but with kid-facing words first and technical
terms second.

## First Lesson Concepts

- say something with `print()`
- remember a value with a variable
- change a value
- repeat with a loop
- choose with an `if`
- use a function machine

## Child-Facing Tone

- short
- warm
- encouraging
- specific
- never shaming
- never overly noisy

## Boundaries

The first version is not a general IDE, not a production sandbox, and not a
free-form AI tutor. It is a focused learning app for tiny Python missions.
