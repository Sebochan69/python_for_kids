# Product Definition

## What It Is

Python for Kids V1 is a beginner coding app where children learn Python by
running tiny quests and seeing what their code does in plain language.

## Target Learners

- Primary: ages 8-10
- Future: ages 5-7 with more guided editing and less typing

## Learning Philosophy

Children should see code as cause and effect:

```python
x = 1
x = x + 1
print(x)
```

Becomes:

1. Python remembers `x` as `1`.
2. Python changes `x` to `2`.
3. Python prints `2`.

The app teaches real Python, but with kid-facing words first and technical
terms second.

## V1 Learning Concepts

- `print()` as making Python print something
- words and numbers as different value types
- variables as memory boxes
- updates as changing a memory box
- math with simple operations
- word joining with text
- comparisons as questions about values
- `if` as a choice/question
- `for` loops as repeaters
- lists as groups of items
- functions as small machines
- checkpoints that mix already introduced concepts

## V1 Product Shape

- header with current quest number and compact Quest List
- mission card with goal, topic, and learning goals
- one-file Python Code Box
- Python Printed output section
- Adventure Log story cards
- code-line highlighting from story cards
- gentle validation summary
- Sebot guide with local explanations and hints

## Child-Facing Tone

- short
- warm
- encouraging
- specific
- never shaming
- never overly noisy

## Boundaries

V1 is not a general IDE, not a production sandbox, and not a free-form AI tutor.
It is a focused learning app for tiny one-file Python quests.
